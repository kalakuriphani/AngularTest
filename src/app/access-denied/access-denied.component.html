You have attempted to access the Open Banking Product data manager, and you are not currently authorised. Please arrange access by raising a Group Service Request (GSR). Approval by your line manager is required.
Description
The Open Banking Product Data Manager is a tool to allow product data to be input to the Open Banking APIs. These APIs allow third party developers to access detailed product information. The APIs can be examined at developer.hsbc.com.
Intended usage
This software is intended for product managers and product teams working with products that need to be exposed via the Open Banking APIs.
Cost
There is no cost for using this software.
Restrictions/exceptions
You must be connected to the HSBC network to gain access. You can be authorised as a Maker or an Approver.
A Maker can create and modify products for publishing to the Open Banking APIs.
An Approver can approve the publishing of products, and is usually a Product Owner.
